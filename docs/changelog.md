## [0.1.0] - 2026-01-09

### Added
- Initial release of Learnsy
- YouTube playlist import functionality
- Video progress tracking with auto-completion at 30% watch time
- Real-time dashboard stats updates
- GitHub-style streak calendar
- Notes and bookmarks for videos
- Keyboard shortcuts (C, B, N, P, L)
- Dark mode support
- Mobile-responsive design
- Clerk authentication with Google OAuth
- Neon PostgreSQL database integration
- Comprehensive README and documentation
- Contributing guidelines
- Code of Conduct
- Security policy
- GitHub issue templates
- Pull request template
- CI/CD workflow with GitHub Actions
- MIT License

### Features
- **Smart Progress Tracking**: Videos auto-complete at 30% watch time
- **Real-time Updates**: All stats refresh instantly via custom events
- **Streak System**: Track daily learning consistency
- **Course Management**: Up to 4 playlists per user
- **Study Interface**: Dedicated video player with notes and bookmarks
- **Dashboard Analytics**: Overview of progress, streaks, and courses

### Technical
- Built with Next.js 14 App Router
- TypeScript for type safety
- Tailwind CSS for styling
- shadcn/ui components
- Server Actions for secure API calls
- Serverless PostgreSQL with Neon
- Deployed on Vercel
